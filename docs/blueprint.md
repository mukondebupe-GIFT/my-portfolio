# **App Name**: GravityStack

## Core Features:

- Physics Sandbox Engine: Integration of Matter.js to simulate 'Anti-Gravity' effects, allowing modular skill chips and project cards to float, collide, and react to drag gestures.
- Triggered Gravity Shift: An interactive animation sequence that transitions a clean, static layout into a dynamic physics-based environment after a 1.5-second delay.
- AI Content Assistant Tool: A built-in AI tool to help the user summarize technical documentation and experience into the professional summary block based on bulleted inputs.
- Fixed Navigation UI: A stable header and footer system that remains pinned and unaffected by the physics simulation to ensure permanent site accessibility.
- Modular Competency Chips: Interactive floating skill elements representing technical domains like OOP and Systems Documentation, allowing for easy textual modification.
- Academic & Project Grid: Clean, highly legible card layout for displaying transcripts and development projects, linked to a localized image asset structure.
- Mobile-Responsive Boundary System: Real-time viewport boundary detection that scales physics containment perfectly for both desktop and mobile resolutions.

## Style Guidelines:

- Primary Color: Deep Navy Blue (#0A192F) for headings and branding, symbolizing authority and precision.
- Background Color: Crisp, pure White (#FFFFFF) to provide a stark canvas for the floating elements.
- Accent Color: High-contrast Slate (#2D3748) for body text and descriptive blocks to ensure perfect digital readability.
- Body and Headline font: 'Inter' used exclusively to maintain a professional, contemporary, and highly scannable look across all devices.
- Modern grid architecture with a 20mm spatial padding and high-contrast separators for modular content blocks.
- Strict physics-based animations where blocks fall to the bottom viewport boundary when left alone, mimicking Google's gravity experiment behavior.
- Clean, minimalist vector iconography located within the skill chips and project cards to visually represent technical categories.
